Termy is a GPU-accelerated, configurable terminal emulator that runs natively on macOS, Windows, and Linux.

## Highlights

- GPU-accelerated rendering with instant startup
- One TOML-style config file with full control
- Native binaries, no Electron, ~18MB memory
- Built-in themes plus a simple custom theme format
- Splits, tabs, search, and optional tmux multiplexing
