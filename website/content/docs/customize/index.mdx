Termy reads configuration from `~/.config/termy/config.txt`. Every setting in Settings has a 1:1 plain-text key you can edit by hand.

<Callout type="info">
  Tmux integration is opt-in. Set `tmux_enabled = true` to start in tmux mode by default. Use the **Manage tmux Sessions** command at runtime to attach, switch, and create sessions.
</Callout>
