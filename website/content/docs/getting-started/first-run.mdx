On first launch, Termy boots into a default theme and writes its config file on demand.

## Defaults

- Theme: `termy`
- Shell: your login shell (`$SHELL`)
- Font: JetBrains Mono at 14px
- Cursor: block, blinking

## Opening Settings

- macOS: `Cmd ,`
- Linux / Windows: `Ctrl ,`

Settings writes back into the same config file used by the CLI.

## Command Palette

- macOS: `Cmd P`
- Linux / Windows: `Ctrl P`

Search every action, theme, and setting. Toggle with `command_palette_show_keybinds` to hide shortcut badges.
