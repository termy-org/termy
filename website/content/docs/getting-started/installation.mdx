---
title: Installation
description: Install Termy on macOS, Linux, or Windows.
---

Termy ships pre-built binaries for macOS, Linux, and Windows. You can also build from source.

## macOS

Download the latest `.dmg` from [Releases](https://github.com/lassejlv/termy/releases) and drag Termy into `/Applications`.

## Linux

Download the AppImage or `.tar.gz` from [Releases](https://github.com/lassejlv/termy/releases).

## Windows

Download the `.msi` installer from [Releases](https://github.com/lassejlv/termy/releases).

## From source

Requires Rust stable.

```bash
git clone https://github.com/lassejlv/termy
cd termy
cargo run --release -p termy
```
