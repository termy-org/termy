Open with `secondary-p` (Cmd on macOS, Ctrl elsewhere).

## What's in it

- Every configurable [action](/docs/keybindings/actions)
- Every theme — instant preview on hover
- Every setting from [Configuration](/docs/configuration)
- Named [tasks](/docs/configuration/tasks)
- tmux session management when `tmux_enabled = true`

## Shortcut badges

Each entry shows its current keybind. Toggle with:

```txt
command_palette_show_keybinds = false
```

## Simple mode

`simple_mode = true` disables the command palette entirely and routes Settings to opening the raw config file.
