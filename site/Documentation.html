Introduction

Termy is a modern, lightweight terminal emulator built from the ground up in Rust. It prioritizes speed, minimalism, and a great out-of-the-box experience.

Key Highlights

  • GPU-accelerated rendering for smooth performance.
  • Minimal resource footprint.
  • Fully customizable via a simple configuration file.
  • Native support for Windows, macOS, and Linux.

Installation

Choose the installation method that fits your operating system.

Windows

Download the .exe or .msi installer from the download section.

macOS

Use the .dmg file provided in our releases. Support for Homebrew is coming soon.

Linux

Run the following command in your current terminal to install Termy instantly:

curl -fsSL https://raw.githubusercontent.com/lassejlv/termy/main/scripts/install-linux.sh | bash

Configuration

Termy uses a single configuration file located at ~/.config/termy/config.toml (or environment specific path). Here you can adjust everything from font family to background opacity. 

[window]
opacity = 0.9
blur = true

[font]
family = "Geist Mono"
size = 14.0
